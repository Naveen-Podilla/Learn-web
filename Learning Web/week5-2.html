JavaScript Basics

1. Variables

Variables in JavaScript can be declared using var, let, or const. The modern approach is to use let and const for better scoping and immutability.

Variable Declaration Example

let name = 'Alice';
const age = 25;

console.log(name); // Output: Alice
console.log(age);  // Output: 25

Output:

Alice
25

2. Functions

Functions in JavaScript can be declared using the function keyword or as arrow functions. Functions are blocks of code designed to perform a task.

Function Declaration Example

function greet(name) {
    return \`Hello, \${name}!\`;
}

console.log(greet('Alice')); // Output: Hello, Alice!

Arrow Function Example

const add = (a, b) => a + b;

console.log(add(5, 3)); // Output: 8

Output:

Hello, Alice!
8

3. Events

Events are actions that occur in the browser, such as clicks or key presses. JavaScript can handle these events using event listeners.

Event Handling Example

<button id="myButton">Click Me!</button>

<script>
    document.getElementById('myButton').addEventListener('click', () => {
        alert('Button was clicked!');
    });
</script>

4. DOM Manipulation

The DOM (Document Object Model) represents the structure of a web page. JavaScript can be used to manipulate this structure dynamically.

DOM Manipulation Example

<div id="myDiv"></div>

<script>
    const myDiv = document.getElementById('myDiv');
    myDiv.innerHTML = '<h2>Hello, World!</h2>';
</script>

5. Introduction to ES6 Features

ES6 (ECMAScript 2015) introduced several new features to JavaScript, including arrow functions, classes, and modules.

Arrow Functions

const greet = name => \`Hello, \${name}!\`;

console.log(greet('Bob')); // Output: Hello, Bob!

Output:

Hello, Bob!

Classes

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        return \`Hello, my name is \${this.name} and I am \${this.age} years old.\`;
    }
}

const person = new Person('Alice', 30);
console.log(person.greet()); // Output: Hello, my name is Alice and I am 30 years old.

Output:

Hello, my name is Alice and I am 30 years old.

Modules

Modules allow you to split your code into separate files and import/export functionalities as needed.

module1.js

export const greet = name => \`Hello, \${name}!\`;

main.js

import { greet } from './module1.js';

console.log(greet('Charlie')); // Output: Hello, Charlie!

Output:

Hello, Charlie!
