HTML5 & CSS3 Guide

1. HTML5 Structure

HTML5 is the latest version of the HTML standard. It introduces new elements and attributes to enhance web page structure and functionality.

Basic HTML5 Document Structure

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        <header>
            <h1>Welcome to HTML5</h1>
        </header>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
        <main>
            <section>
                <h2>Section Heading</h2>
                <p>This is a paragraph of text.</p>
            </section>
        </main>
        <footer>
            <p>© 2024 Your Company</p>
        </footer>
    </body>
</html>

Output:

<header>
    <h1>Welcome to HTML5</h1>
</header>
<nav>
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
<main>
    <section>
        <h2>Section Heading</h2>
        <p>This is a paragraph of text.</p>
    </section>
</main>
<footer>
    <p>© 2024 Your Company</p>
</footer>

2. CSS3 Properties

CSS3 introduces new properties and features to enhance web design. Some of the most common properties include:

Basic CSS3 Syntax

/* CSS Comment */
body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 0;
    padding: 0;
}

h1 {
    color: #0056b3;
}

Output:

<body>
    <h1>Styled Heading</h1>
</body>

CSS3 Box Model

The CSS3 Box Model consists of the following components:

  • Content: The actual content of the box.
  • Padding: Space between the content and the border.
  • Border: The border surrounding the padding (if any).
  • Margin: Space outside the border.
.box {
    width: 200px;
    padding: 20px;
    border: 2px solid #0056b3;
    margin: 10px;
}

Output:

<div class="box">
    Content inside the box
</div>

3. Responsive Design with Flexbox

Flexbox is a one-dimensional layout method for aligning items within a container. It allows items to grow, shrink, and align easily.

Basic Flexbox Example

<div class="flex-container">
    <div class="flex-item">Item 1</div>
    <div class="flex-item">Item 2</div>
    <div class="flex-item">Item 3</div>
</div>

.flex-container {
    display: flex;
    justify-content: space-around;
    background: #f4f4f4;
    padding: 20px;
}

.flex-item {
    background: #0056b3;
    color: white;
    padding: 20px;
    border-radius: 4px;
    text-align: center;
    width: 30%;
}

Output:

Item 1
Item 2
Item 3

Responsive behavior:

  • On screens narrower than 768px, items stack vertically.
  • On screens narrower than 480px, items take full width.

4. Responsive Design with Grid

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with ease.

Basic Grid Example

<div class="grid-container">
    <div class="grid-item">Item 1</div>
    <div class="grid-item">Item 2</div>
    <div class="grid-item">Item 3</div>
</div>

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    background: #f4f4f4;
    padding: 20px;
}

.grid-item {
    background: #0056b3;
    color: white;
    padding: 20px;
    border-radius: 4px;
    text-align: center;
}

Output:

Item 1
Item 2
Item 3

Responsive behavior:

  • On screens narrower than 768px, the grid changes to 2 columns.
  • On screens narrower than 480px, the grid changes to a single column.
