C# Basics

1. Variables

In C#, a variable is used to store data that can be referenced and manipulated in a program. Variables must be declared with a specific data type.

// Variable declaration and initialization
int age = 25;
string name = "John Doe";
double height = 5.9;

// Output to the console
Console.WriteLine("Name: " + name);
Console.WriteLine("Age: " + age);
Console.WriteLine("Height: " + height);

2. Data Types

C# supports various data types that can be broadly classified into value types and reference types.

Value Types

  • int: Represents integer values.
  • double: Represents floating-point numbers.
  • char: Represents a single character.
  • bool: Represents a boolean value (true or false).

Reference Types

  • string: Represents a sequence of characters.
  • object: The base type for all other types.
// Example of data types
int number = 42;
double price = 19.99;
char initial = 'A';
bool isCSharpFun = true;
string greeting = "Hello, C#!";

// Output to the console
Console.WriteLine("Number: " + number);
Console.WriteLine("Price: " + price);
Console.WriteLine("Initial: " + initial);
Console.WriteLine("Is C# Fun?: " + isCSharpFun);
Console.WriteLine("Greeting: " + greeting);

3. Control Structures

Control structures allow you to control the flow of execution in your program.

Conditional Statements

// Example of if-else statement
int number = 10;

if (number > 0)
{
    Console.WriteLine("Number is positive.");
}
else if (number < 0)
{
    Console.WriteLine("Number is negative.");
}
else
{
    Console.WriteLine("Number is zero.");
}

Loops

Loops are used to execute a block of code multiple times.

// Example of a for loop
for (int i = 0; i < 5; i++)
{
    Console.WriteLine("Value of i: " + i);
}

// Example of a while loop
int j = 0;
while (j < 5)
{
    Console.WriteLine("Value of j: " + j);
    j++;
}

4. Collections

Collections in C# are used to store groups of related objects. The most commonly used collections are arrays, lists, and dictionaries.

Arrays

// Example of an array
int[] numbers = { 1, 2, 3, 4, 5 };

foreach (int number in numbers)
{
    Console.WriteLine("Number: " + number);
}

Lists

Lists are dynamic arrays that can grow and shrink in size.

// Example of a List
List names = new List { "Alice", "Bob", "Charlie" };

names.Add("Diana");
names.Remove("Bob");

foreach (string name in names)
{
    Console.WriteLine("Name: " + name);
}

Dictionaries

Dictionaries store key-value pairs.

// Example of a Dictionary
Dictionary ageDictionary = new Dictionary
{
    { "Alice", 30 },
    { "Bob", 25 },
    { "Charlie", 35 }
};

foreach (KeyValuePair entry in ageDictionary)
{
    Console.WriteLine("Name: " + entry.Key + ", Age: " + entry.Value);
}
